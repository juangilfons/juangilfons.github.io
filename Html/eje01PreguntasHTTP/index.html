Cuestionario 1

Preguntas sobre criptografía y capas de seguridad

  • Plantear un caso de uso de encriptación unidireccional

    Un caso de uso es el almacenamiento de contraseñas en bases de datos. Se utiliza un algoritmo de hash, como SHA-256, para convertir la contraseña en una cadena de texto irreversiblemente encriptada.

  • Cuáles son las ventajas y desventajas de la encriptación asimétrica?

    Ventaja: Permite la encriptación segura sin necesidad de compartir una clave secreta. Es ideal para la firma digital.

    Desventaja: Es más lenta que la encriptación simétrica y requiere mayor capacidad de procesamiento.

  • Como encriptaríamos para obtener confidencialidad en la transferencia de datos?

    Encriptamos con la clave privada, el receptor decripta con la publica.

  • Como encriptaríamos para autenticar un documento usando firma electrónica?

    Utilizar encriptación asimétrica. Se genera un hash del documento y se encripta con la clave privada del firmante.

  • Que tipo de encriptación se usa en la WEB?

    Se usa encriptación simétrica (AES) en combinación con encriptación asimétrica (RSA) para el establecimiento de conexiones seguras mediante HTTPS.

Cuestionario 2

Preguntas sobre HTTP

  • Sobre que protocolo binario está montado el protocolo HTTP?

    HTTP se basa en el protocolo TCP.

  • Cuáles son los clientes http y los servidores http mas utilizados?

    Clientes: Navegadores como Chrome, Firefox, Safari.

    Servidores: Apache, Nginx, IIS.

  • Que verbos admite un comentado en el requerimiento HTTP?

    GET, POST, PUT, DELETE y PATCH

  • Que contenido lleva el body de un requerimiento HTTP?

    Puede incluir datos enviados en un formulario, JSON, XML, archivos binarios, entre otros.

  • Que diferencia existe entre un URL y un URI?

    Un URL (Uniform Resource Locator) es un tipo específico de URI (Uniform Resource Identifier) que indica cómo acceder a un recurso específico, incluyendo su localización.

  • Como se almacena información relacionada con las respuestas HTTP?

    A través de cookies, almacenamiento local (localStorage), o almacenamiento de sesión (sessionStorage).

  • Que significa Virtual hosting?

    Técnica que permite alojar múltiples dominios (sitios web) en un único servidor.

  • Porque el HTTP se considera STATE LESS?

    Porque cada petición es independiente y no mantiene ningún estado entre transacciones.

  • Que nueva version de HTTP se está usando para mejorar la velocidad de la WEB?

    HTTP/2.

  • Como va a mejorar el comportamiento de una aplicación WEB en ambientes ruidosos o de débil conexión?

    HTTP/3 utiliza QUIC, un protocolo basado en UDP que mejora la latencia y la velocidad de las conexiones, especialmente en redes con pérdida de paquetes o alta latencia.
